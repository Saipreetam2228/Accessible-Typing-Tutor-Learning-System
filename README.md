# Inclusive Typing Tutor – Vidya Vahini

An interactive typing tutor developed as part of the Sri Sathya Sai Vidya Vahini Inclusive Education initiative. This project is designed to help children, including differently-abled learners, improve their keyboard typing skills through structured levels, visual guidance, and engaging feedback mechanisms.

The system focuses on accessibility, simplicity, and gamified learning to ensure that students of all abilities can practice typing in a supportive environment.

## Key Features

• Multi-level typing practice modules  
• Words Per Minute (WPM) tracking  
• Typing accuracy measurement  
• Interactive visual feedback  
• Accessibility-friendly design for inclusive education  
• Keyboard guidance for left and right hand usage  
• Support for alternative typing methods (e.g., hunt-and-peck typing)  
• Audio and visual cues for engagement  

## Technology Used

• HTML5  
• CSS3  
• JavaScript  

## Project Structure

The application is organized into multiple learning levels where students progressively improve their typing speed and accuracy.


## Purpose

This project aims to support inclusive education by providing an accessible typing learning platform that can be easily deployed in schools and educational programs.

## Future Improvements

• User login and progress tracking  
• Achievement badges and gamification  
• Typing analytics dashboard  
• Expanded accessibility features  
• Adaptive difficulty levels
